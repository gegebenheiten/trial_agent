Table: R_Arm_Study

1) AE_Spec
- Definition: Percentage of participants with adverse events of special interest (AESI).
- Value_Type: Percentage_String
- Output: “NN%” / “NN.N%” / “0%” / “Unknown”
- Decision rules:
  - Only fill when the text explicitly labels events as “AEs of special interest”, “AESI”, or equivalent.
  - If explicit percent reported → return it.
  - Else compute using Protocol A3 if AESI count AND denominator (group N) are explicit.
  - If explicitly states no AESI → “0%”.
  - Otherwise → “Unknown”.
- Example:
  - “AEs of special interest: 1%” → “1%”

2) Assess_Adherence
- Definition: Proportion of participants who completed key assessments or endpoint evaluations as planned.
- Value_Type: Percentage_String
- Output: “NN%” / “NN.N%” / “0%” / “Unknown”
- Decision rules:
  - Only fill when the source explicitly reports completion/compliance for assessments/evaluations/endpoints.
  - Accept phrases like: “assessment adherence”, “assessment compliance”, “completed endpoint evaluation”, “completed all evaluations”.
  - If explicit percent → return it.
  - Else compute using Protocol A3 if numerator and denominator are explicit for this group.
  - If explicitly states none completed → “0%”.
  - Otherwise → “Unknown”.

3) Completer
- Definition: Proportion of participants who completed the study (completed trial).
- Value_Type: Percentage_String
- Output: “NN%” / “NN.N%” / “0%” / “Unknown”
- Decision rules:
  - Prefer explicitly reported completion percentage.
  - Otherwise compute ONLY if BOTH Completed and Started counts for the current group are explicitly given:
    Completed / Started * 100 (use Protocol A3, with denominator = Started).
  - If explicitly states nobody completed → “0%”.
  - If denominator is unclear → “Unknown”.

4) Cycle_Length

- Value_Type: Integer (days)
- Output: integer (e.g., 21) / “Unknown”
- Decision rules:

  - If “X-week cycle” is stated, convert to days = 7 * X.
  - If “28-day cycle” is stated, return 28.
  - If not stated → “Unknown”.

9) Dosage_Level
- Definition: Dosage level(s) for interventions in the current arm/group.
- Value_Type: Text_String
- Output: dose string(s) EXACTLY as in text, de-duplicated; join multiple with “; ”; or “Unknown”
- Decision rules:
  - Return dose strings as stated (keep units; do not convert).
  - If multiple interventions: prefix each dose with intervention name:
    “DrugA: 10 mg/kg; DrugB: 5 mg”
  - If dose not explicitly stated → “Unknown”.

10) Dosing_Frequency
- Definition: Frequency of administration (normalized tokens).
- Value_Type: Text_String (normalized schedule tokens)
- Output: token(s) joined with “; ”, possibly prefixed by intervention name; or “Unknown”
- Allowed tokens: QD, BID, TID, QW, Q2W, Q3W, Q4W
- Decision rules:
  - Only fill when schedule is explicitly indicated.
  - Normalize mapping:
    daily/once daily → QD
    twice daily → BID
    three times daily → TID
    weekly → QW
    every 2 weeks → Q2W
    every 3 weeks → Q3W
    every 4 weeks / every 28 days → Q4W
  - If multiple interventions, prefix each frequency with the intervention name.
  - If unclear → “Unknown”.

15) Group_Type
- Definition: Label describing the role/type of the current trial arm.
- Value_Type: Categorical
- Allowed values (including “unable to judge”):
  - Experimental
  - Active Comparator
  - Placebo Comparator
  - Sham Comparator
  - No Intervention
  - Other
  - Unknown
- Decision rules:
  - Scope: Decide ONLY for THIS arm. Prefer arm-scoped structured fields and arm↔intervention linkage.
  - 1) Use structured arm_group_type if provided for THIS arm.
  - 2) Else use explicit keywords in group_title_raw/group_desc_raw:
       Sham Comparator: “sham”, “simulated procedure”, “mock”
       Placebo Comparator: “placebo”, “matched placebo”, “vehicle”, “saline placebo”
       No Intervention: “no intervention”, “observation only”, “watchful waiting”, “no treatment”, “natural history”
  - 3) Else infer using study object ONLY if arm-scoped linkage is unambiguous:
       - Placebo/vehicle/saline linked intervention → Placebo Comparator
       - Sham/simulated procedure linked intervention → Sham Comparator
       - No linked intervention AND explicit observation-only arm text → No Intervention
  - 4) Experimental vs Active Comparator:
       - Experimental: explicit investigational/study-specific intervention for THIS arm (e.g., “study drug”, “investigational”, novel code-name).
       - Active Comparator: explicit marketed/approved therapy clearly used as comparator/benchmark for THIS arm.
       - If ambiguous which arm gets which, return “Unknown”.
  - 5) Single-arm studies:
       - If clearly investigational → Experimental
       - If clearly observation/no treatment → No Intervention
       - If marketed therapy without investigational signal and without comparator framing → Other
  - 6) Conflict handling:
       - Prefer most specific comparator signals: Sham > Placebo > No Intervention
       - For remaining conflicts, if cannot resolve by arm-specific evidence → “Unknown”
  - 7) Do NOT use masking/allocation alone.
  - If still uncertain → “Unknown”.
