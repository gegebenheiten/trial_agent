Feature notes (definitions):
Baseline_Severity_Score
Type: Text (scale name + threshold / empty)
Definition: Specific baseline disease severity score thresholds required for eligibility.
Extraction rule: Return TEXT only if a named severity scale AND an explicit threshold are stated. Preserve the original wording exactly, including comparison operators and qualitative modifiers (e.g., “HAM-D > 20”, “PASI ≥ 12”, “CGI-S ≥ 4”, “baseline severity score above moderate”). Do NOT infer thresholds. Do NOT normalize, calculate, or restate. If disease severity is mentioned without a specific scale or threshold (e.g., “severe disease”), return empty.

Cardio_Func
Type: Catalog (yes / no / empty)
Definition: Cardiac (heart) function criteria specified in eligibility.
Extraction rule: Return "yes" if TEXT explicitly mentions or clearly implies cardiac function requirements, cardiovascular disease exclusions, or heart-related thresholds (e.g., LVEF cutoffs, heart failure, arrhythmia). Return "no" if TEXT explicitly states that no cardiac-related eligibility restrictions apply or clearly limits eligibility criteria to non-cardiac systems. Otherwise return empty.
